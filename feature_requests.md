# Feature Requests

## add card pickup to the game log

in the setup phase (and probably later on), when a player picks up a card, it would be nice to have that logged in the game log. This would help players keep track of what cards have been picked up and by whom. 


## allowing refreshing without leaving the game

currently, if a player refreshes the page, they are taken back to the lobby. I haven't tested what happens if they re-enter the same game code mid-game. It would be nice if players could refresh the page without being taken back to the lobby, if they are in the middle of a game. This would allow them to fix any issues they might be having without losing their progress in the game.

## allowing all players to bid on a proposed trade prior to the trade being accepted

currently, when a player proposes a trade, all players can see and accept the trade. this means a quick player could swoop in and accept a trade not meant for them. I think we should have several trading modes: to a specific player, or to all players. if a trade is proposed to a specific player, only that player can accept the trade. if a trade is proposed to all players, then any player can accept the trade. the original trade proposer then needs to confirm the trade with the accepting player of their choice before the trade is executed.
