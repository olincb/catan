# Feature Requests


## allowing all players to bid on a proposed trade prior to the trade being accepted

currently, when a player proposes a trade, all players can see and accept the trade. this means a quick player could swoop in and accept a trade not meant for them. I think we should have several trading modes: to a specific player, or to all players. if a trade is proposed to a specific player, only that player can accept the trade. if a trade is proposed to all players, then any player can accept the trade. the original trade proposer then needs to confirm the trade with the accepting player of their choice before the trade is executed.

## escape the "reconnecting..." screen

when reconnecting, it might sit there for a while. give the player an option just to exit back to the main lobby screen.

## do not shorten "Dev Card"

I don't like "Dev" instead of "Development". please use only the fully spelled out version. keep in mind the previous bug about the "Development Card" text overlapping with its cost in the build cost section.

## update the shape of a settlement

right now, settlements are just triangles. it would be great if they were more of a house shape, with a roof. I haven't seen the cities yet, but I imagine their shape could use some work too.

## small "beach" border around the board

right now, the colored tiles go right up to the water. it would be nice to have a small yellow-beige border around the board to give it a more "beachy" feel. this would also help visually separate the land from the water.

## random board setup means unfair configurations are possible

do some research online about some rules or guidelines for avoiding too-powerful positions. I found a situation where there were 2 6's and an 8 all next to each other, giving a very strong intersection on the board. I think there should be some rules to prevent this from happening, such as not allowing more than one 6 or 8 to be adjacent to each other. this would help create a more balanced and fair game. 

## do not allow entering road/settlement/city place mode when the player does not have the resources to build that item

It's probably better to prevent a player from entering build mode for an item at all, if they won't be able to afford it. maybe we can disable the buttons to place a road/settlement/city if the player doesn't have the resources to build that item, and also have a tooltip mentioning as much.

## sheep/pasture tile color

right now, the sheep/pasture tile is a mint color. it should be a more natural green color, to better represent grass and pastures. the current mint color is a bit too bright and doesn't fit well with the overall aesthetic of the game.
