# Feature Requests


## tooltips on build buttons

the tooltips on the build buttons have good content, but it would be nice to use a proper component for them rather than relying on the browser's default title attribute. it would also be nice if there was no delay for the tooltip to appear, and if the tooltip could be styled to match the rest of the game.

## show card count in the game log when bank trades are placed

when bank trades are placed, the game log currently just shows that it happened. it would be nice to show exactly which cards were traded, and how many of each card were traded.

## larger font for dots on tokens

the dots indicating how likely a token is to be rolled are a bit too small. let's keep the concept the same but make the dots larger.
